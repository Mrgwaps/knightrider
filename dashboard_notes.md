# KITT Dashboard Reference Analysis

## Overall Layout
- Black background throughout
- Wide horizontal panel, roughly 4:1 aspect ratio
- Divided into distinct sections left-to-right

## Section 1: FAR LEFT PANEL (leftmost column)
- Vertical column of small colored LED dots (red/yellow/green)
- "GUIDANCE" label with cyan/teal bar indicator (0-100)
- "EXIST.RDY" label with cyan/teal bar indicator (0-100)  
- Large red 7-segment display: "0000.0" (5 digits with decimal)
- Below: row of small green LED dots
- Bottom buttons (stacked vertically):
  - Green "5" button + orange "P AUX" button
  - Orange "SKI MODE" button + orange "LASER" button
  - Green "AUTO ROOF L" button + orange "MICRO JAM" button
  - Gray "SILENT MODE" button
  - Red "EJECT L" button + gray "LIC PLATE" button

## Section 2: MPH DISPLAY PANEL
- Scale markings: 0 to 200 (top bar)
- Red LED bar graph (horizontal, speed indicator)
- Large red 7-segment: "000" MPH
- Below: row of small yellow/green LED dots
- Row of small rectangular green LED indicators (about 12)

## Section 3: CENTER-LEFT CONTROL PANEL
- Top buttons (yellow/orange):
  - "POWER" (yellow)
  - "MIN RPM" (yellow)  
  - "FUEL ON" (yellow)
  - "IGNITORS" (yellow/orange)
- Main button grid (2 columns x 4 rows):
  - Left col: AIR (yellow), OIL (yellow), P1 (orange), P2 (orange)
  - Center: AUTO CRUISE (orange), NORMAL CRUISE (orange), PURSUIT (blue)
  - Right col: S1 (orange), S2 (orange), P3 (orange), P4 (orange)
- Between columns: tall vertical red LED bar graph (voice synthesizer)

## Section 4: RPM GAUGE (center)
- Arc/sweep gauge: 0-100 scale with red arc
- Numbers: 100, 75, 50, 25, 10, 5 (curved)
- Red needle indicator
- Large red 7-segment: "00" RPM display
- Below gauge: multiple horizontal bar graphs with labels:
  - INLET TEMP (0-100)
  - MASS FLOW LBS (0-200)
  - E.G.T. °F (0-2000)
  - FUEL FLOW GPH (30-200)
  - FREE TURBINE (0-?)
  - MAIN OIL PRESS (0-300)

## Section 5: RIGHT-CENTER PANEL
- Top: vertical column of colored LED dots
- Labels with bar graphs:
  - TURBO BOOST
  - 6 RM (green button)
  - 7 DLA (green button) + P ENG (orange button)
  - 8 PL1 (green button)
  - GRPLO MODE (green button) + OIL SLICK (orange button)
  - P IND (green button) + EJECT R (red button)
- AUTO ROOF R button

## Section 6: SENSOR/STATUS PANEL
- Multiple horizontal bar graphs with labels:
  - COMPRESSOR TEMP °F (0-500)
  - HYD GYGT STRESS PSI (0-200)
  - VOLTS DC (0-32)
  - CAPACITY STATUS (FULL indicator)
  - PROPAGATION DELAY HRS (0-30)
  - ACCESS (0-400)
- APPROACH WARNING SEC (0-90)
- RADAR WATTS KW (0-?)
- SENSOR RANGE (0-20)
- EJECT L / EJECT R buttons (red)
- SURVEILLANCE MODE label
- CANCEL button

## Section 7: RIGHT PANEL - STATUS BARS
- FUEL GALLONS (0-50)
- MI/GALLONS (0-60)  
- RANGE ESTIMATES (0-100)

## Section 8: FAR RIGHT - CAMERA SCREENS
- Two large cyan/teal rectangular screens with rounded corners
- Dark gray bezels/frames
- Screens appear to show camera feeds (solid teal in reference)
- Below screens: COMPUTER PRINTOUT label with white rectangle
- Bottom right: 4 red square buttons labeled: POWER, AUTO, NORM, PURSUIT

## Key Visual Style
- All on pure black background
- Button colors: yellow, orange, red, green, blue, gray
- LED indicators: red, yellow, green dots
- Bar graphs: red/green gradient fills
- 7-segment displays: bright red
- Camera screens: cyan/teal
- Text: white, small caps
- Overall feel: 1980s sci-fi computer aesthetic
